# Renpy Android Game Localization Tool

A Python script tool for localizing Renpy Android games, supporting both rooted and non-rooted devices.

## ✨ Features

- 🔍 Automatically scan game text resources
- 📝 Extract text into an editable format (JSON)
- 🔄 Auto-translate while the game is running
- 📱 Persistent local storage — no need for frequent translation API calls

## 📦 Requirements
- **Android Device**: Android 5.0+

### 3. Configure API Key
Edit `rpy`:
```json
{
  "api_provider": "tencent",
  "api_key": "Your Tencent Translation API Key",
  "source_lang": "en",
  "target_lang": "zh"
}
```

## 🚀 Usage

### Method 1: Rooted Device
1. **Inject localization**:
   - Move the generated `*.rpy` files to:
     `/data/app/package_name/game/`

### Method 2: Non-Rooted Device (MT Manager)
1. Open the game APK using MT Manager
2. Use the "Inject File Provider" feature
3. After creating local storage, move the localization files into the `game/` directory

### Custom Dictionary
```json
{
  "character_names": {
    "John": "John",
    "Mary": "Mary"
  },
  "special_terms": {
    "HP": "Hit Points",
    "MP": "Mana Points"
  }
}
```
